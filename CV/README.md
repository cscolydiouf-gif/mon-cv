# 📄 CV Interactif - Cheikh Sadibou Coly Diouf

## 📌 Description

Ce projet est un CV numérique développé en **HTML**, **CSS** et **JavaScript**. Il présente mon profil, mes compétences, ma formation, mes expériences et mes centres d'intérêt dans une interface moderne, responsive et interactive.
